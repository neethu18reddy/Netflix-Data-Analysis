# GitHub Project Description

## Short (one-liner, for repo "About")
End-to-end EDA of 8,807 Netflix titles in Python — pandas cleaning, 15 business
questions, 11+ matplotlib/seaborn/plotly charts, and a Streamlit dashboard.

## Medium (repo description paragraph)
A portfolio-grade data analysis of Netflix's Movies and TV Shows catalogue. The project
walks through a real-world analytics workflow — inspecting and cleaning the raw
`netflix_titles.csv` (8,807 rows, 12 columns), engineering time and multi-value features,
and answering 15 concrete business questions such as *"which genres are growing fastest?"*
and *"which months see the most content additions?"*. Deliverables include an executed
Jupyter notebook, a modular `src/` package (cleaning, analysis, visualisation), 11+
professional charts saved to `images/`, and an interactive Streamlit dashboard.
