# Resume Bullet Points — Netflix Data Analysis Project

Copy-paste ready. Pick the 3–5 strongest for your CV.

* Cleaned and analysed **8,807 Netflix titles across 122 countries and 42 genres** using
  Python and pandas, engineering date/duration/multi-value features to unlock 15 business
  questions.
* Built a **modular data-cleaning pipeline** (rename, standardise, impute, deduplicate,
  parse dates, split multi-value columns) reused by both the Jupyter notebook and a
  Streamlit dashboard.

  Movies + Dramas fastest-growing genres**.

  country, content type) and 6 interactive Plotly charts.

  chart followed by 3–5 business insights, and future-work recommendations included.
