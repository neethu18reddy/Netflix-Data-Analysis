# LinkedIn "Projects" Description

**Netflix Movies and TV Shows — End-to-End Data Analysis**

I built a portfolio-grade data analytics project on Netflix's public catalogue of 8,807
titles across 122 countries and 42 genres to demonstrate a real analyst workflow —
from messy CSV to executive dashboard.

**What I did**
• Cleaned and standardised a messy real-world dataset in Python and pandas — handled
  missing values, split multi-value country/genre/cast columns, parsed dates, and
  engineered new features.
• Answered 15 concrete business questions such as *movies vs TV shows split, top
  producing countries, fastest-growing genres, monthly seasonality of content additions,
  and typical movie runtime.*
• Built 11+ professional visualisations with matplotlib, seaborn, and Plotly, plus an
  interactive Streamlit KPI dashboard with dynamic filters.
• Documented every insight with a business-analyst interpretation and packaged the code
  into a modular `src/` Python package.

**Key insights**
1. Movies make up ~70% of the catalogue; TV shows ~30%.
2. The US, India, and UK together produce more than 50% of all Netflix titles.
3. TV-MA is the most common rating — Netflix is an adult-skewing platform.
4. 2019 was peak content-addition year; growth has plateaued since.
5. International Movies and Dramas are the fastest-growing genres.

**Tools**
Python · Pandas · NumPy · Matplotlib · Seaborn · Plotly · Streamlit · Jupyter
